React, Tailwind and SCSS

At The Movies

At the Movies is a website that displays information on the latest popular, top rated and upcoming movies. The site allows for toggling between light and dark mode, and it offer search capabilities.

At the Movies was written using React and Tailwind CSS. It uses various React techniques such as UseEffect, UseState, importing/exporting classes, and others. The site also performs a third-party API call to retrieve the movie data and posters.

.Net, SQL Server, SSIS & SSRS

Owners Capital

Owners Capital is an application that imports loan data from other applications, exports data to the enterprise financial application, and it calculates the owners equity in terms of stock and cash.

The application is written in VB.Net for the front-end and middle tier, and SQL Server for the database. Stored procedures are used for CRUD operations. SSIS is used to import/export data. SSRS is used to display various reports for the end-users.

I support this application at my current company, as such, a demo is not possible, and accesss to the code is not possible.
